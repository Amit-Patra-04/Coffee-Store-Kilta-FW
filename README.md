## 🌟 Features

- **Modern & Responsive Design:** Adapts beautifully to mobile, tablet, and desktop screens.
- **Dynamic Animations & Interactions:**
  - **Scroll Effects:** Navigation bar styling updates on scroll, and elements smoothly animate into view as you scroll down the page.
  - **Hover Effects:** Interactive menu items, testimonials, and buttons that scale and cast shadows on hover.
  - **Custom Cursor Follower:** A unique aesthetic cursor that follows your mouse and highlights when hovering over clickable elements.
  - **Ripple Effect:** Click interactions generate a subtle ripple effect across the page.
  - **Parallax Scrolling:** The hero section background moves at a different speed for a 3D depth effect.
  - **Steam Animation:** Simulated steam rising dynamically in the hero area.
- **Sections:**
  - **Home (Hero):** Eye-catching introductory section with parallax background.
  - **About:** The story, mission, and journey of the coffee brand.
  - **Menu:** Showcase of premium coffee blends (featuring Kilta Coffee Co. products) with direct order links.
  - **Testimonials:** Reviews from satisfied coffee lovers.
  - **Contact:** A styled contact form for user inquiries.
- **Sticky Header & Mobile Navigation:** Easy access to different sections with a smooth-scrolling hamburger menu on mobile devices.

## 🛠️ Technologies Used

- **HTML5:** Semantic structure and content layout.
- **CSS3:** Styling, Flexbox/Grid layouts, custom properties (variables), and keyframe animations.
- **Vanilla JavaScript:** DOM manipulation, event handling, scrolling effects, and custom interactive behaviors (built without external frameworks).
- **Boxicons:** Iconography used in the footer.

## 🚀 Getting Started

This is a static website, so no complex setup or build process is required to view it.

1. **Clone the repository** (or download the files).
2. **Open `index.html`** in any modern web browser (Chrome, Firefox, Safari, Edge).
   - Alternatively, you can use a local development server like VS Code's "Live Server" extension for the best viewing and development experience.

## 📁 File Structure

- `index.html`: The main HTML document containing the structure of the webpage.
- `style.css`: The stylesheet containing all CSS rules, variables, animations, and responsive media queries.
- `main.js`: The JavaScript file powering the interactive elements, form validation mock, and scroll behaviors.
